git常用命令

  • 同步远程分支(清理无效的远程追踪分支)：git remote prune origin --dry-run

  • 删除本地分支： git branch -d branch_name

  • 创建远程分支：

    • git checkout -b branch_name
    • git push origin branch_name:branch_name
    • git branch --set-upstream-to=origin/branch_name
